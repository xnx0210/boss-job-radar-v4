# 🎯 口播文案 · lakejobai-job-radar

**时长**：~120秒 ｜ **语速**：220字/分 ｜ **风格**：技术产品介绍


【00:00-00:10 痛点】

BOSS直聘上找工作，每天要搜几十个关键词、翻几百个岗位、换十几个城市——手动点"立即沟通"点到手软，HR的回复还得盯着手机回。

这不是求职，这成了体力活。

【00:10-00:30 产品引入】

所以我们做了一个工具，叫 lakejobai-job-radar，一个 AI 驱动的 BOSS 直聘自动化控制台。

它的定位很明确：你来定策略，它来跑腿。

【00:30-00:55 核心功能】

打开 Web 控制台，选个城市、输个关键词、点搜索，几十个岗位就出来了。

然后点击"一键投递"，它会自动逐个打开岗位详情页，发送你来定好的招呼语，投递完成后，页面实时显示进度条，可以随时取消。

这还不是最关键的——关键是投递之后的事。

【00:55-01:15 AI 聊天】

投递成功之后，平台后台会持续监控聊天。一旦有 HR 回复，AI 就会自动接管——根据你预先配置的个人资料和简历摘要，生成自然的回复内容，发过去。

如果 HR 要简历、微信号、手机号，系统会通过 BOSS 官方按钮自动发送，及时加上一句"已发送，请查收"。

整个过程你只需要看，不需要动手。

【01:15-01:40 CLI + AI Agent 集成】

除了 Web 控制台，我们还提供了完整的命令行工具。装完项目之后，直接运行 lakejob 命令，14 条命令覆盖搜索、投递、聊天、分析、候选池管理全流程。

关键是：所有命令输出都是结构化的 JSON，非常方便接入 AI Agent。

你可以让 Claude、open-code、Cursor、Copilot、GPT——任何大模型来调用它。

Agent 先跑一下 lakejob schema，拿到 12 个工具的描述，然后根据用户意图自动调用——搜岗位、投递、看聊天、AI分析匹配度，一条命令搞定。

相当于你有了一个 AI 求职管家。

【01:40-01:55 多模型支持】

AI 模型支持 DeepSeek V4 Pro、OpenRouter、小米 MiMo 等平台，也可以填任意 OpenAI 兼容 API。

设置页选平台，自动填 Base URL 和模型列表，填一个 Key 就行。
